# jhd-ai-workflows

Reusable AI workflow assets, prompts, and operating instructions.

## Structure

- `core-instructions/`: foundational instruction files such as `AGENTS.md` and `CLAUDE.md`
- `skills/`: reusable skill definitions and workflow-specific guidance

## Notes

This repo starts with core instruction files and can expand over time with additional workflow assets, templates, and process guidance.

The repository currently includes an MIT license for permissive reuse.
